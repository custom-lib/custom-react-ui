简介

基于 Tippy 的悬浮弹框组件。

内容

通过设置 Content 属性设置弹框内容。Content可为任意合法的ReactNode。

如果Content是字符、数字，默认容器样式将会是灰底白字。如果是JSX Element或者组件，则容器将只提供动画，不包含任何样式。

<Popper Content="abcdef">
  <button>字符</button>
</Popper>
<Popper Content={123}>
  <button>数字</button>
</Popper>
<Popper Content={<span style={{ backgroundColor: 'red' }}>JSX.Element</span>}>
  <button>JSX Element</button>
</Popper>
<Popper
  Content={() => (
    <div
      style={{
        backgroundColor: 'lightgreen',
        width: 180,
        height: 60,
        color: 'red',
      }}
    >
      React.Component
    </div>
  )}
>
  <button>React 组件</button>
</Popper>

容器样式

通过 class, style 属性设置弹框容器的样式。

<Popper
  Content="这是一个按钮"
  style={{ backgroundColor: 'red', padding: 12 }}
>
  <button>Button</button>
</Popper>

箭头

通过设置 arrow 属性为 true 启用弹框箭头。 箭头的颜色与弹框背景色相同，但是没有边框。可以通过覆盖Popper下.tippy-arrow的样式修改。

<Popper Content="这是一个按钮" arrow>
  <button>arrow</button>
</Popper>
<Popper Content="这是一个按钮" arrow style={{ backgroundColor: 'red' }}>
  <button>arrow</button>
</Popper>
<Popper
  Content="这是一个按钮"
  arrow
  style={{ backgroundColor: 'green', border: '1px solid red' }}
>
  <button>arrow</button>
</Popper>

动画持续时间

通过设置 animationDuration 属性(ms)，修改弹框进出场动画持续时间。

type animationDuration = number | { enter: number, leave: number }
<Popper Content="这是一个按钮" animationDuration={500}>
  <button>Button</button>
</Popper>
<Popper
  Content="这是一个按钮"
  animationDuration={{ enter: 200, leave: 1000 }}
>
  <button>Button</button>
</Popper>

进出场延时

通过设置 delay 属性(ms)，修改弹框进出场延时。

type delay = number | { enter: number, leave: number }
<Popper Content="这是一个按钮" delay={500}>
  <button>Button</button>
</Popper>
<Popper Content="这是一个按钮" delay={{ enter: 200, leave: 2000 }}>
  <button>Button</button>
</Popper>

触发方式

通过设置 trigger 属性为 "mouseenter focus" | "click" | "focusin" | "mouseenter click" 设置弹框触发方式。

默认为 "mouseenter focus"。

也可以不设置 trigger 属性， 通过设置 visible 属性为 true / false 手动控制弹框显示 / 关闭。

<Popper Content="这是一个按钮" trigger="mouseenter focus">
  <button>mouseenter focus</button>
</Popper>
<Popper Content="这是一个按钮" trigger="click">
  <button>click</button>
</Popper>
<Popper Content="这是一个按钮" trigger="focusin">
  <button>focusin</button>
</Popper>
<Popper Content="这是一个按钮" trigger="mouseenter click">
  <button>mouseenter click</button>
</Popper>
<Popper Content="这是一个按钮" visible={true}>
  <button>manual</button>
</Popper>

点击隐藏

通过设置 hideOnClick 属性为 boolean | "toggle" 设置弹框在 trigger = "click" 下时的隐藏方式。

true 时点击任意地方隐藏， false 时点击不触发隐藏， "toggle" 时点击触发元素自身隐藏。

默认为 true。

<Popper Content="这是一个按钮" trigger="click" hideOnClick={true}>
  <button>true</button>
</Popper>
<Popper Content="这是一个按钮" trigger="click" hideOnClick={false}>
  <button>false</button>
</Popper>
<Popper Content="这是一个按钮" trigger="click" hideOnClick="toggle">
  <button>toggle</button>
</Popper>

通过设置 followCursor 属性为 boolean | "horizontal" | "vertical" | "initial" 设置弹框跟随鼠标移动的方式。

默认为 false。

<Popper Content="这是一个按钮" followCursor>
  <button>true</button>
</Popper>
<Popper Content="这是一个按钮" followCursor={false}>
  <button>false</button>
</Popper>
<Popper Content="这是一个按钮" followCursor="initial">
  <button>initial</button>
</Popper>
<Popper Content="这是一个按钮" followCursor="horizontal">
  <button>horizontal</button>
</Popper>
<Popper Content="这是一个按钮" followCursor="vertical">
  <button>vertical</button>
</Popper>

可交互的

通过设置 interactive 属性为 true 使得鼠标移动可移动至弹框上。

通过设置 interactiveBorder 属性(px) 设置可交互弹框的边界。

通过设置 interactiveDebounce 属性(ms) 设置超出可交互弹框的边界多少时长时隐藏弹框。

<Popper Content="这是一个按钮" interactive>
  <button>interactive</button>
</Popper>
<Popper Content="这是一个按钮" interactive interactiveBorder={50}>
  <button>interactiveBorder=50</button>
</Popper>
<Popper Content="这是一个按钮" interactive interactiveDebounce={666}>
  <button>interactiveDebounce=666</button>
</Popper>

禁用状态

通过设置 disabled 属性为 true 禁用弹框。

type delatriggerTargety = someElement | [someElement1, someElement2, ...];

通过设置 triggerTarget 属性为合法的DOM Element改变弹框附着对象。
